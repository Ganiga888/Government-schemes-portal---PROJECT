# Government Schemes Portal

## 📌 Project Overview
This project is a web application that helps users explore and apply for various government schemes.

## 🚀 Features
- User Registration & Login
- Browse Government Schemes
- View Scheme Details
- Apply for Schemes
- Notifications System
- Chatbot Assistant
- Dark Mode & Multi-language Support

## 🛠️ Technologies Used
- Frontend: HTML, CSS, Bootstrap, React
- Backend: Node.js / Express
